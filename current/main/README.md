# Main Resume

## Purpose

This resume is designed for **broadest appeal**, targeting as many areas and industries as possible. It is the default resume given out when the exact job description or field is unknown.

## When to Use

- Distributing through extended social networks
- General job applications where the specific role isn't clear
- Networking events where you meet people from various industries
- Situations where you want to cast a wide net rather than target a specific position

## Strategy

This version emphasizes:
- Versatile skills that transfer across industries
- Broad professional experience
- Adaptability and diverse capabilities

Rather than optimizing for a specific role, this resume aims to generate interest and start conversations across multiple potential career paths.
