# Full-Stack AI Startup Resume

## Purpose

This resume is designed for **full-stack web development roles at AI-native consumer startups**, targeting early-to-mid stage companies building AI-powered web applications. Positions the candidate as a founder-minded engineer who can take products from 0→1.

## When to Use

- Full-Stack Engineer at AI-native startups
- Founding Engineer / Early Technical Hire
- Frontend Engineer at AI startups (leverage AI integration experience)
- Backend Engineer at AI startups (leverage LLM integration + web backend)
- AI Product Engineer roles
- Technical Co-founder opportunities

## Target Profile

**Company Type:** Early-to-mid stage startups (seed to Series B)  
**Product Type:** Consumer AI web applications (SaaS, productivity, creative tools)  
**Key Needs:** 0→1 product building, React/Next.js expertise, AI integration, full-stack ownership

## Strategy

This version emphasizes:
- **0→1 Product Building**: Technical co-founder experience taking Tumble from concept to MVP
- **Modern Full-Stack**: Next.js 14, TypeScript, React Server Components, Tailwind CSS
- **AI Integration**: OpenAI API, prompt engineering, streaming responses, multi-step AI workflows
- **Backend Depth**: FastAPI, Node.js, PostgreSQL, microservices architecture
- **Startup Velocity**: 4-month MVP, same-day deployments, rapid iteration with user testing
- **Founder Mindset**: Full ownership, architectural decisions, cross-functional coordination

The focus is on demonstrating rare combination of **React/Next.js frontend expertise** + **AI/LLM integration** + **0→1 product building experience** that AI-native startups desperately need.

## Key Differentiators

1. **Full-Stack AI**: Can build AI features end-to-end (UI → API → LLM → response)
2. **Founder Mindset**: Comfortable with ambiguity, rapid iteration, wearing multiple hats
3. **Production AI**: Real LLM integration at scale (not just side projects)
4. **Modern Stack**: Current-generation web technologies (Next.js 14, App Router, RSC)
5. **Shipped Products**: Actual user-facing products in production
